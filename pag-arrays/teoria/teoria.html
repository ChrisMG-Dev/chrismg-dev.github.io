Teoría de arrays

Creación de arrays

Se pueden crear de dos formas

  1. var array = new Array();
  2. var array = [];

Ambas formas crean un array vacío

Si se define de la siguiente forma: var array = new Array(10);
Se creará un array con un tamaño de 10, pero de elemento vacíos [ , , , , , , , , , , ]

Algo parecido ocurre si hacemos lo siguiente:
var array = []; array[2] = "hola";
Resultado: [ , ,"hola"]

Eliminar elementos de un array

array.length; // 8
delete array[2];
array[2]; // undefined

Borrar un elemento de array no libera la memoria. Con el operador delete simplemente lo reemplaza por undefined

Se recomendable utilizar el método splice en su lugar

var a = ["a","b","c","d"];
a.splice(1,3); // ["a"];

Otra forma es la siguiente:
a.length = 0; // []

Añadir elementos a un array

Los elementos de añaden normalmente de la siguiente manera:
array[indice] = "elemento";

El inconveniente es que si nuestra intención es añadir al final del array, debemos conocer cual es el siguiente índice
Se puede hacer de la siguiente manera:
array[array.length] = elemento;

Para añadir al final de un array, existe un método push
array.push(elemento);

Recorrido de un array

La forma mas común de recorrer un array de la siguiente:
for (i = 0;i < array.length; i+= 1) {
    document.write(array[i]);
}

Existe otra forma más elegante de hacer esto

for each (var manzana in bolsaManzanas) {
    document.write(manzana);
}

Tipos de arrays

Arrays paralelos - Vectores

Son los mas simples y comunes, son los que hemos visto hasta ahora:
var a = [1, 2, 3, 4];
a[1]; // 2

Arrays Multidimensionales

Son arrays que contienen a su vez arrays

Ejemplo:
var alumnos = [["Pepe","Gomez"],["María","Pérez"]]
alumnos[0][1] // Gomez

Métodos

Concat

Permite concatenar un array con otro

var a = ["h","o"];
var b = ["l", "a"];
a.concat(b); // ["h","o","l","a"]

Sort

Ordena los elementos de un array

var a = [3,6,1,2];
a.sort() // [1,2,3,6];

Reverse

Invierte un array

var a [1,2,3];
a.reverse(); // [3,2,1]
